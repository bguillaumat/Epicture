# EPICTURE
Epicture is a non-official Imgur Application in Java (Android)
## Features:
* **Login/Disconnect Imgur**
* **Upload photos from files or camera**
* **Manage Likes**
* **Manage photos**
* **Comments**
* **SearchBar**
